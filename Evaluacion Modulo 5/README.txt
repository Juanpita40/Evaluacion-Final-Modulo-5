Nuestro  proyecto  tiene como objetivo entregar una  solucion  a  la problematica de cada  empresa  en  particular
en este  caso  es  poder  organizar bajo  un solo  sistema los servicios   entregados  a  los  clientes,    en  el 
ambito de canalizar la informacion, planificar oportunamente actividades con clientes, altas y bajas de clientes y
usuarios identificacion de nuestros profesionales, reportes, revision de documentos.
Proporcionamos un sistema de informacion amigable y realmente adaptado a sus necesidades.

Contenido:
		
Crear_usuario.jsp		Permite crear un usuario en sistema.
crearcapacitacion.jsp		Formulario para agendar una capacitación en el sistema. .
editaradministrativo.jsp	Permite modificacion de datos usuario/perfil Administrativo.
editarcapacitacion.jsp		Permite modificacion de datos usuario/perfil Capacitación.
editarcliente.jsp		Permite modificacion de datos usuario/perfil Cliente.
editarprofesional.jsp		Permite modificacion de datos usuario/perfil Profesional
Inicio.jsp			Presentacion de nuestro sistema.
listado_de_usuarios.jsp		Crea usuario segun perfil de Cliente, Administrativo, Profesional.
Listarcapacitaciones.jsp	Listado de capacitaciones registradas.
login_usuario.jps		Permite ingreso al listado de usuario.
loginsesion.jsp			Permite crear sesion.
mantencion.jsp			Permite mostrar sitios aun no implementados.
sesioncreada.jsp		Nos entrega movilidad en sitio.
contacto.jsp 	 		Formulario de contacto para realizar consultas y solicitudes.
